## 垂直切片（实施顺序）

### 切片 1: 基础与配置
**目标**: 搭建项目基础设施
- Types（类型定义）: prompt.ts, tag.ts, file.ts, ui.ts
- Utils（工具函数）: markdown.ts, date.ts, storage.ts
- Constants（常量）: config.ts, keybindings.ts, defaults.ts
- Build config（构建配置）: vite, tsconfig, tailwind

### 切片 2: 文件系统基础
**目标**: 实现本地文件访问
- FileService: showDirectoryPicker, readFile, writeFile, listFiles
- fileStore: 持久化目录句柄
- useDirectoryPicker hook: 目录选择钩子
- Directory selection UI: 目录选择界面

### 切片 3: 数据层
**目标**: 读取和解析 Prompt 文件
- MarkdownUtil: 解析 MD，提取 frontmatter
- PromptService: CRUD 操作
- TagService: 解析和组织标签
- promptStore, tagStore: 状态存储

### 切片 4: UI 基础
**目标**: 核心布局结构
- CSS 变量: 从原型提取
- Common components: Button, IconButton, Modal, Overlay
- Layout components: Sidebar, TopBar shell, DetailPanel shell
- uiStore: UI 状态管理

### 切片 5: Prompt 展示
**目标**: 以网格形式展示 Prompt
- PromptCard component: Prompt 卡片组件
- PromptGrid component: 卡片网格布局
- TagPill component: 标签胶囊
- DetailPanel: 详情面板（只读视图）
- PromptContent: Markdown 渲染

### 切片 6: Prompt 创建
**目标**: 创建新 Prompt
- CreateModal component: 新建/编辑弹窗
- TagSelect component: 标签选择器
- PromptService.create(): 创建方法
- File saving: 文件保存

### 切片 7: Prompt 编辑与删除
**目标**: 修改现有 Prompt
- Edit mode: CreateModal 的编辑模式
- Delete confirmation: 删除确认
- History versioning: 历史版本（.history/ 目录）
- Real-time save: 实时保存

### 切片 8: 搜索与导航
**目标**: 快速查找 Prompt
- SearchService: FlexSearch 封装
- SearchInput component: 搜索输入框
- TagTree component: 分层标签树
- Filtering logic: 筛选逻辑

### 切片 9: 命令面板
**目标**: 快速访问 (Cmd+K)
- CommandPalette component: 命令面板组件
- Keyboard shortcuts: useKeyboardShortcuts 快捷键钩子
- Quick actions: 快速操作

### 切片 10: 批量操作与导出
**目标**: 批量操作
- Batch selection UI: 批量选择界面
- Batch delete: 批量删除
- ExportService: JSON, CSV 导出
- Export UI: 导出界面

## 风险缓解

| 风险 | 缓解措施 |
|------|----------|
| File System Access API 不支持 | 显示友好错误，建议使用 Chrome/Edge |
| 大量文件时的性能问题 | 虚拟滚动、懒加载 |
| 浏览器权限被拒绝 | 清晰的说明、优雅降级 |
| 并发编辑冲突 | 文件锁、冲突检测 |
| 数据丢失 | 历史版本、回收站目录 |

## 完成定义

每个切片完成的标准:
1. 所有组件/存储/服务已实现
2. TypeScript 编译无错误
3. 功能在浏览器中正常工作
4. 代码符合风格指南
5. 无控制台错误或警告
